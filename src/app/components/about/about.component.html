Demonstration Shopping Cart Application - created with Angular

This is a demonstration shopping Cart illustrating :

  • Component based Structure
  • Services used for non-related components
  • Child/Parent communications are directive based
  • Utilises Angulars state based templating
  • Image data retrieved dynamically from a third party API (unsplash.com)
  • Observables are used for services
  • Typescript from Frontend to Backend!
  • Interfaces with a RESTful API Backend, which uses
    • Node
    • Express
    • Postgres as Database
    • JSON Web Token (JWT) for authentication
    • Middleware for admin, user verification on API routes
    • API routes (setup with Postman)
    • Safe SQL retrieval

There are various functions integrated, for example:

  • Register unique users (API)
  • Show all completed orders for logged in users (API)
  • Show items in completed orders (API)
  • Maintain a persistant cart (API)
  • Adds items to a cart (API)
  • Removes items from a cart (API)
  • Adds items to a cart (API)
  • Technologies used, ngBootstrap/toasts/postgres/etc
